== Description ==
Tracktum is Woocommerce Conversion plugin for Google, Facebook, Pinterest.
Tracktum inserts Javascript codes on WooCommerce cart page, checkout success page and after user registration. So you can track who are adding your products to cart, who are purchase them and who are registering to your site.


== Features ==
* Add to Cart Event
* Add to Wishlist Event
* Purchase Event
* Registration Event
* Wishlist Event
* Product View Event
* ViewCategory Event
* Search Event
